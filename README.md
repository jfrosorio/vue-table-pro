# vue-table-pro

A Vue component for rendering tables from JSON data.

## Usage
### NPM
```
npm install vue-table-pro
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
